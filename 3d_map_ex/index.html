cartography

a series of examples around creative cartography in 2D and 3D, using THREE.js and custom tools.

you'll need a local server with PHP to run the demos and load some tilesets (elevations especially)

the source code is available on a github repo https://github.com/nicoptere/cartography and licensed under MIT

2D tile systems

  • using a .mbtiles file to store tiles for offline use
  • cutting a big picture into XY tiles

2D vectors

  • basic GPS data: plotting a month of personal data collected from my google Dashboard
  • clustering GPS data: using alpha shapes to cluster data visually
  • aggregating GPS data: plotting a day of 5 random San Francisco cabs GPS data (source)
  • filtering GPS data: removing noise and indexing colors on the length of each segment

(stupid use of) 2D tiles rasterd tiles

  • plotting the locations of some of lisbon's azuljeos as a voronoi diagram
  • flying over volcanoes
  • detecting faces on map

3D elevation

the following demos use a tileset to compute the elevations of the vertices of a plane

it's based on http://www.elasticterrain.xyz elevation tileset

this will not work online

  • basic elevation
  • rendering
  • normals
  • custom shader
  • post processing ( cheap Depth Of Field )

3D city

the following demos use https://openstreetmap.us/~migurski/vector-datasource/ vector information to generate 3D models on the fly

  • drawing 2D lines in 3D
  • triangulating vector shapes
  • extruding the triangulated sections
  • improving the build time by merging geometries
  • adding custom attirbutes to the merged geometry
  • viszualizing the city of San Fransisco
